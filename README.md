# AgroInsight - Inteligência Geoespacial para Safras

O **AgroInsight** é uma plataforma web desenvolvida para transformar dados de geolocalização em decisões estratégicas para o produtor rural. Através da integração de APIs climáticas e processamento de dados agronômicos, o sistema acompanha o ciclo do cultivo desde o planejamento até a colheita.

---

## Funcionalidades Principais

### Gestão de Talhões e Geolocalização
* **Mapeamento de Áreas:** Permite ao usuário delimitar suas áreas de plantio via coordenadas geográficas (Latitude/Longitude).
* **Cadastro de Safras:** Registro detalhado da cultura (ex: Soja, Milho, Trigo), variedade da semente e data de semeadura.

### Painel de Insights Inteligentes
* **Monitoramento Climático:** Insights em tempo real sobre precipitação, temperatura e umidade baseados na localização exata da fazenda.
* **Ciclo Fenológico:** Estimativa automática dos estágios de crescimento da planta (Vegetativo/Reprodutivo) usando cálculos de graus-dia.
* **Alertas de Risco:** Notificações sobre janelas críticas de geada, estresse hídrico ou calor excessivo.

### Planejamento e Produtividade
* **Calendário Agrícola:** Sugestões personalizadas de datas para adubação, aplicação de defensivos e colheita.
* **Análise de Performance:** Comparativo entre a produtividade estimada no planejamento e a produtividade real colhida.

- [ ] **Calculadora de Insumos:** Cálculo automático de sementes e fertilizantes com base na área (hectares).
- [ ] **Mapeamento de Pragas:** Registro de focos de doenças com "pin" geolocalizado no mapa.
- [ ] **Modo Offline:** Aplicativo PWA para registro de dados em áreas sem conectividade.
